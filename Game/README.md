##### Structure

- Game
  - entry file
- Main
  - initialize canvas and global objects
  - initialize sprites and bind click events
- Director
  - take hand of the logic part of the game
  - manage the creation or destroy
- Resources
  - Manage all the image resources
- Resource Loader
  - Loader resources
- Sprites
  - Background.js / Land.js 
  - Score / StartButton
  -  UpPencil / DownPencil
  - Bird
